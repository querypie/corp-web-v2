# Overview

높은 수준으로 데이터베이스 접근 및 보안을 관리하기 위해 역할과 속성을 기반으로 한 다양한 형태의 액세스 제어를 제공합니다.
인증된 사용자에 한해 QueryPie를 통해 접근 권한이 부여된 데이터베이스에만 액세스할 수 있으며 SQL 실행은 미리 정의된 SQL 구문 조합을 기반으로 실행이 제한됩니다.

# Granting Access Control Privileges

STEP 1 **Privilege** 메뉴에서 `Create Privilege Type` 버튼을 눌러 권한을 생성합니다.

- **Name:** Privilege를 구분할 수 있는 이름
- **Database Type:** 구문 속성에 따라 **General, Redis**로 구분됩니다.
- **Privilege Type:** 해당 권한으로 실행을 허용할 구문을 선택합니다.
- **Functions:** 해당 권한으로 허용 가능한 작업을 선택합니다.
- QueryPie Web Editor 내에서의 기능을 제어합니다.

STEP 2 **Access Control** 메뉴에서 권한을 부여할 사용자 혹은 그룹을 선택합니다.

STEP 3 권한을 부여할 대상 커넥션을 찾고 Privilege를 선택합니다.
